# TCP/UART对齐界面应用

这是一个优化后的TCP/UART统一通信应用，实现了设置界面和数据显示界面的完美对齐，提供更直观和对称的用户体验。

## 🎯 界面优化特点

### 1. 完美对齐设计
- **垂直对齐**：TCP设置界面与TCP数据显示界面垂直对齐
- **垂直对齐**：UART设置界面与UART数据显示界面垂直对齐
- **左右对称**：TCP和UART区域左右对称分布
- **独立完整**：每个区域都包含完整的设置和显示功能

### 2. 界面结构
```
┌─────────────────────────────────────────────────────────┐
│                TCP/UART统一通信应用                      │

│      TCP通信        │           UART通信                │
│  ┌───────────────┐  │  ┌─────────────────────────────┐   │
│  │ 模式选择      │  │  │ 串口配置                   │   │
│  │ 连接配置      │  │  │ 数据发送                   │   │

│  ├───────────────┤  │  ├─────────────────────────────┤   │
```

## 🚀 主要功能

### 1. TCP通信区域（左侧）
- **设置界面**：
  - 模式选择（客户端/服务端）
  - 连接配置（IP地址和端口）
  - 数据发送（文本输入和快速发送）
  - 换行符控制
- **数据显示**：
  - TCP数据接收显示
  - 清空TCP显示按钮
  - 自动滚动功能

### 2. UART通信区域（右侧）
- **设置界面**：
  - 串口配置（串口选择和波特率）
  - 数据发送（文本输入和快速发送）
  - 换行符控制
- **数据显示**：
  - UART数据接收显示
  - 清空UART显示按钮
  - 自动滚动功能

## 📋 使用方法

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 运行应用
```bash
python tcp_uart_unified_app.py
```

### 3. TCP通信使用
1. 在左侧TCP区域选择客户端或服务端模式
2. 配置连接参数（IP地址和端口号）
3. 点击连接/启动服务
4. 在发送框中输入数据并发送
5. 在下方TCP数据显示区域查看接收数据

### 4. UART通信使用
1. 在右侧UART区域点击"刷新端口"检测可用串口
2. 选择串口和波特率
3. 点击"连接串口"
4. 在发送框中输入数据并发送
5. 在下方UART数据显示区域查看接收数据

## 🔧 技术实现

### 1. 界面布局
- **QSplitter**：水平分割器，左右分布TCP和UART
- **QVBoxLayout**：垂直布局，上下分布设置和显示
- **QGroupBox**：功能分组，界面更清晰

### 2. 数据显示
- **独立显示**：TCP和UART数据在各自区域显示
- **自动滚动**：新数据自动滚动到底部
- **独立控制**：支持独立的数据显示控制

### 3. 多线程架构
- **TCP线程**：处理TCP通信
- **UART线程**：处理串口通信
- **信号槽机制**：线程间安全通信

## 💡 使用技巧

### 1. 数据监控
- 左侧查看TCP数据，右侧查看UART数据
- 可以同时监控两种通信方式的数据
- 使用自动滚动保持最新数据可见

### 2. 数据管理
- 使用"清空TCP显示"清空TCP数据
- 使用"清空UART显示"清空UART数据
- 可以分别管理不同类型的数据

### 3. 换行符控制
- TCP发送时可选择是否添加换行符
- UART发送时可选择是否添加换行符
- 适应不同协议需求

## 🚨 注意事项

1. **串口权限**：Linux系统可能需要串口权限
2. **端口占用**：确保端口未被其他程序占用
3. **数据编码**：目前只支持UTF-8编码
4. **界面大小**：建议使用较大屏幕以获得最佳体验
5. **对齐布局**：界面布局对齐，需要足够的屏幕空间
