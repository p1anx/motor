# TCP/UART应用 - 三页面结构

## 功能概述

已成功重构应用为三页面结构，实现了功能分离和独立导航。用户可以在主页面选择功能，然后进入对应的专门页面进行操作。

## 🎯 页面结构

### 四个独立页面
1. **主页面 (索引0)** - 功能选择界面
2. **TCP通信页面 (索引1)** - 完整的TCP通信功能
3. **UART通信页面 (索引2)** - 完整的UART通信功能
4. **同步控制页面 (索引3)** - 同步配置和控制功能

### 界面布局
```
┌─────────────────────────────────────────────────────────────────────────────┐
│ 导航栏: 应用标题 + TCP通信 + UART通信 + 同步控制 + 状态指示器          │
├─────────────────────────────────────────────────────────────────────────────┤
│ 页面0: 主页面 - 功能选择界面                                              │
│         ┌─────────────┐ ┌─────────────┐ ┌─────────────┐                │
│         │ TCP通信   │ │ UART通信    │ │ 同步控制   │                │
│         │ (蓝色)    │ │ (红色)      │ │ (橙色)     │                │
│         └─────────────┘ └─────────────┘ └─────────────┘                │
├─────────────────────────────────────────────────────────────────────────────┤
│ 页面1: TCP通信页面 - 完整的TCP通信功能                                    │
│ 页面2: UART通信页面 - 完整的UART通信功能                                 │
│ 页面3: 同步控制页面 - 同步配置和控制功能                                │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 🔄 导航方式

### 1. 主页面导航
- **TCP通信按钮** (蓝色) - 进入TCP通信页面
- **UART通信按钮** (红色) - 进入UART通信页面
- **同步控制按钮** (橙色) - 进入同步控制页面

### 2. 导航栏导航
- **应用标题** - 返回主页面
- **TCP通信** - 快速切换到TCP页面
- **UART通信** - 快速切换到UART页面
- **同步控制** - 快速切换到同步页面

### 3. 页面内导航
- **返回主界面按钮** - 各页面返回主界面

## 📊 状态显示

### 主页面状态
- **连接状态**: 显示TCP/UART连接状态
- **同步状态**: 显示同步进度和状态

### 导航栏状态
- **同步状态指示器**: 实时显示同步状态

### 状态颜色编码
- **未连接/未启用**: 灰色 `#7f8c8d`
- **TCP连接**: 蓝色 `#3498db`
- **UART连接**: 红色 `#e74c3c`
- **同步启用**: 橙色 `#f39c12`
- **同步完成**: 绿色 `#27ae60`

## 🔧 技术实现

### 核心组件
- **QStackedWidget**: 四页面堆栈管理
- **TCPCommunicationPage**: TCP通信页面类
- **UARTCommunicationPage**: UART通信页面类
- **SyncControlPage**: 同步控制页面类

### 页面管理
```python
# 页面索引
0: 主页面
1: TCP通信页面
2: UART通信页面
3: 同步控制页面

# 页面切换
self.stacked_widget.setCurrentIndex(index)
```

### 状态同步机制
- **主应用状态**: 核心状态管理
- **页面状态**: 各页面状态显示
- **指示器更新**: 导航栏状态实时更新
- **数据共享**: 页面间数据同步
